# Código de Conduta

## Nosso compromisso

Como mantenedores e contribuidores, nos comprometemos a tornar a participação neste projeto uma experiência livre de assédio para todos, independentemente de idade, corpo, deficiência, etnia, identidade e expressão de gênero, nível de experiência, nacionalidade, aparência, raça, religião ou orientação sexual.

## Nossos padrões

Exemplos de comportamentos que contribuem para um ambiente positivo:

- Ser respeitoso e empático.
- Aceitar feedback construtivo com maturidade.
- Focar no melhor para o projeto e para a comunidade.

Exemplos de comportamentos inaceitáveis:

- Linguagem ofensiva, ataques pessoais, intimidação ou assédio.
- Trolling deliberado, insultos ou desqualificação.
- Publicar informações privadas de outras pessoas sem permissão.

## Responsabilidade

Os mantenedores do projeto são responsáveis por esclarecer padrões de comportamento e tomar ações corretivas apropriadas quando necessário.

## Escopo

Este código de conduta se aplica em todos os espaços do projeto (issues, PRs, discussões, etc.).

## Reportando problemas

Para relatar incidentes ou comportamento inadequado, envie um e-mail para `irving@ruas.dev.br`.

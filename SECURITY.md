# Segurança

Se você acredita ter encontrado uma vulnerabilidade de segurança neste projeto:

- **Não** abra uma issue pública com detalhes.
- Envie um e-mail para `irving@ruas.dev.br` com o máximo de informações possível (passos, impacto, evidências).

Vamos responder assim que possível e trabalhar para resolver o problema.

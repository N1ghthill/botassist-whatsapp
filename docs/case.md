# Case - BotAssist WhatsApp

## Problema
Times e profissionais que atendem via WhatsApp perdem produtividade com respostas repetitivas, baixa padronização e falta de controles de operação.

## Solução
Aplicativo desktop (Electron) com bot de WhatsApp, IA via Groq, controles de owner, políticas de grupo e fluxo de manutenção/documentação.

## Tech Stack
- Node.js
- Electron
- Baileys
- Groq API

## Status
- `production-like`: pronto para uso real com controles de operação.
- Distribuição via releases para Windows/macOS/Linux.
